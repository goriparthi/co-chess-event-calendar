Colorado Scholastic Chess Events Calendar

This project provides a calendar of scholastic chess events in Colorado, available for download as an iCalendar file.

About

This project scrapes the latest chess events from the Colorado Chess Association website and generates an iCalendar (.ics) file that can be imported into your calendar application.
